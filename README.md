# zkp_video
Video Integrity Verification Method
Copy the C files and the sample videos on your computer in the same folder.
This code is developed in C language and uses gcc compiler. OpenSSL cryptographic library, libsodium and libssl-dev package must be installed.

We conducted experiments on a PC having Intel Core i7-45U CPU@1.8GHz√ó4 and 12 GB RAM. The test videos from different bench mark datasets such as VIRAT, SULFA and Derf's collections and five publicly available video segments from YouTube are used. For each test videos, the experiments run about 30 times and the average value is taken.
